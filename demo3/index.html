browserify-shim使用：

  • 如果"three": "global:THREE",带global说明是使用外链，则three.js则不会合并到bundle.js中，只是引用
  • 如果没有global，则会使用本地指向的three.js文件，会被合并到bundle.js中

如果使用depends依赖的话，则可以在代码中省去require('jquery')，如下：

"browserify-shim":{
"jquery": "$",
"three": "global:THREE",
"test":{ "exports":"Test","depends": [ "jquery" ] }
}

在代码中：

//global.jQuery = require('jquery'); 此处不需要引用jquery
var Test = require('test');
$('body');
Test();
